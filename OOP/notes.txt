Class :- A class is a code template for creation Objects
methods :- Methods are function written inside a class
Objects:- Object is an instance of the Class. Objects are variables that contain data and functions(methods) that can be used to manipulate the data.
Constructor :- A special method that is called when an Objects is created. __init__
self :- It represents the instance of class
Encapsulation :- It is a mechanism of wrapping the data (variables) and code acting on the data (methods) together as a single unit.
